## Usage

```tsx
<Files>
  <Folder name="Folder 1">
    <File name="file1.txt" />
    <File name="file2.txt" />
  </Folder>
  <Folder name="Folder 2">
    <File name="file3.txt" />
    <File name="file4.txt" />
  </Folder>
</Files>
```
