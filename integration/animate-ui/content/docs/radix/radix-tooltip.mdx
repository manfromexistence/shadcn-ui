## Usage

```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger>Tooltip Trigger</TooltipTrigger>
    <TooltipContent>Tooltip Content</TooltipContent>
  </Tooltip>
</TooltipProvider>
```
